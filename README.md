### 🚀 Executando Localmente

1. Clone o repositório
```
git clone https://github.com/link/para/o/repositório
``` 
2. Instalação

```
npm install
```
2. Executando
```
npm start
```

3. O jogo irá rodar na [porta 3000](http://localhost:3000/) no seu navegador.
